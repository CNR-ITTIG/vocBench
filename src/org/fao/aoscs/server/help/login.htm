- This part of the AGROVOC-CS Workbench guide will help you to login.

Step 1:  Enter login information.

New User  Please register using the [register] link in the login page.  
Anonymous User Please check the [Sign in as Anonymous] check box.            
Registered User  Please enter your Username & Password    
Step 2:  Select preferred Language from the [Language] drop down list.

# NOTE:  This language refer to the language of the interface (not the language for the concept
 server data). If you don't select any language, the language selected in you user preference
 if available will be used otherwise English language will be selected by default.

Step 3:  Select preferred ontology from the [Ontology] drop down list.

# NOTE: If you don't select any ontology, the ontology selected in you user preference if available
 will be used otherwise default ontology will be selected.

Step 4:  Check the [Remember me] check box, if you want the browser to remember your login details.

Step 5:  Click [Sign In] button to proceed.

# NOTE:  Login if successful, you will be directed to the initial page set on your user preference
 otherwise default home page.
