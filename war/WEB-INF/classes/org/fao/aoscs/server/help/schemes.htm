Schemes Module

This module is available from the top of every page. It includes classification schemes.

Use this module to view or work with classification schemes.

Choosing the scheme to work with

You can choose which available scheme you wish to view with the drop down menu. By default, the AGRIS/CARIS Classification Scheme is shown.

Viewing options

If you have not already done so, you may wish to change which languages you wish to show when viewing the classifications by going to the "Customize Language Display" button available on the top right of every page. Note: this only changes the number of terms you will see in the left panel, or when you are viewing only the classification. You will always see all terms in all languages available in the right panel when you choose a category.

You can also reduce the number of terms shown in the left panel by choosing the option "Show only preferred terms".

You can also choose whether or not to view the URI.

Working with schemes

To add a new scheme, use the following steps:
  1. Go to the Schemes Module.

  2. Click on the add icon [+] next to the dropdown menu.

  3. Enter the name, namespace, and description.

  4. Click "Submit".


To delete a scheme, use the following steps:
  1. Go to the Schemes Module.

  2. Choose the scheme you wish to delete from the dropdown menu.

  3. Click on the delete icon [-] next to the dropdown menu showing the scheme you want to delete.

  4. Confirm the deletion.


To change the name or description of a scheme, use the following steps:
  1. Go to the Schemes Module.

  2. Choose the scheme you wish to edit from the dropdown menu.

  3. Click on the edit icon next to the dropdown menu showing the scheme you want to edit.

  4. Make your changes and confirm.


To add a new category to a scheme, use the following steps:
  1. Go to the Schemes Module.

  2. Choose the scheme you want to add a category to from the dropdown menu.

  3. Choose a category at the same level you want to create the new category at, or the parent of the new category.

  4. Click on the add icon [+] above the the dropdown menu.

  5. Choose "Create from scratch" if you want to create a new category or "Select an existing concept" to use a concept that already exists in the Workbench

  6. If you choose "Create from scratch", select the language of the new category.

  7. Note whether the new category is at the same level as the category you selected, or a child of the selected category.

  8. Click "Submit".


To delete a category from a scheme, use the following steps:
  1. Go to the Schemes Module.

  2. Choose the scheme you wish to work with from the dropdown menu.

  3. Choose the category you wish to delete

  4. Click on the delete icon [-] above to the dropdown menu showing the scheme you want to delete.

  5. Confirm the deletion.
