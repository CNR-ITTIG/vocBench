Preferences Module

You can use this module to set preferences for future sessions. If you wish to make changes for the current session only, use the "Customize language display" button at the top right of the page.

Passwords

Use this module to change your password using the following steps:
  1. Type a new password in the "New password" field.

  2. Confirm it in the "Confirm password" field.

  3. Save your changes in the bottom right of the page.


Email

Use this module to change your email using the following steps:
  1. Type a new email in the "New email" field.

  2. Confirm it in the "Confirm password" field.

  3. Save your changes in the bottom right of the page.


Language of data

Under "Language of the data" you are viewing all the languages that are included in your current viewing of the data.

Note: If you wish to select multiple languages when adding or removing languages, you can select a group of languages by first selecting one language, and then holding down the Shift key while clicking a second.

You can add languages you view data in (concepts, terms, etc.) by using the following steps:
  1. Click the "Add" button to see a list of languages to add. If no languages appear, then all the languages available are already being viewed.

  2. A "Select language" popout will appear. If a language you wish to see data in appears, you may select it and click the the "Add" button.

  3. The new language will then appear under "Language for the data".

  4. Save your changes in the bottom right of the page.


You can remove languages you view data in (concepts, terms, etc.) by using the following steps:
  1. Under "Language of the data", click on the languages you wish to remove.

  2. Click "Remove".

  3. When asked if you wish to remove the language, click "OK".

  4. Save your changes in the bottom right of the page.

  5. You will no longer see this language under "Language for the data". If you wish to add it again, follow the instructions above for adding a language.


Language of the interface

To change the language you view the interface in, simply select it from "Language of the interface" and save your changes in the bottom right of the page.

Other options

  • You can choose which page you wish to see first when coming to the Workbench by choosing which page you wish to see under "Set your initial page".

  • You can choose whether you want to see URIs and deprecated terms and concepts by clicking or unclicking those check boxes.
