Search

The Search Module allows you to search for concepts, relationships, terms and other information associated with concepts.

The following information is information for the default search page. For more options, see the page for advanced search.

By default, only published terms are searched. If you wish to search for terms with a different status (such as proposed, revised, deleted, etc), use the advanced search.
  1. Type one or more words or a phrase that best describe the information you want to find into the search box.
  2. Select one of the following search filters from the dropdown list on the left:

    • 'Contains' will list all the concepts with terms containing the search term anywhere, even within another term, e.g. searching for 'rice' with the filter 'contains' will also return 'Agricultural prices' and 'Cricetus'.

    • 'Exact match' will list only concepts with terms that exactly match the search term. This choice brings back fewer results than the other choices. Searching 'rice' will ONLY return the concept rice and will exclude 'rice cakes', 'rice flour', 'rice fields', etc.

    • 'Starts with' will list all the concepts that start with the search term you have entered. Searching 'rice' will show you 'rice cakes', 'rice flour', 'rice fields', etc, but not 'wild rice' or 'flooded rice'.
Note: If you wish to see all terms in the system, click the search button without entering any text. (This will return very large results)

Other search options

You also have the following options when searching:
  1. Case sensitivity: by default, searches are not case sensitive (whether a word is capitalized or not does not matter). Choose this box if you would like your search to be case sensitive, such as in searching for 'Turkey' or 'turkey'.

  2. Include description: by default, the search results do not include the description. The description includes definitions and comments.

  3. Include spelling variations: by default, searches include only spelling variations entered as preferred or non-preferred terms. To include spelling variants such as labour and labor, check this option.

Advanced Search

To see advanced search options, click on 'more options' from the main page for the Search Module.

You can combine advanced search options with the default search options.

When you click on 'more options' from the search page you will have the following additional options:
  1. Relationship: to use this function, click the 'Browse' button to choose a relationship. Choose the relationship you are looking for and click the “Close button”. The selected relationship will be shown in the advanced search area. Note: This filter will list only those concepts linked with the specified relationship. If you do not use any search terms in the search box, the results will list all concepts using the selected relationship.

  2. Sub-vocabulary: this option allows you to limit your search to a sub-vocabulary within the Workbench, such as "geographic concepts" or "taxonomic terms". Click on the '+' sign to expand the tree.

  3. Term code: this allows you to search for the numeric code of a term or concept. You may also specify the repository where you would like to search the term, such as AGROVOC, ASFA, etc. To use this option, type the term code into the Term Code box, and if you wish, choose a repository you would like to restrict your search to.

  4. Status: you may specify a status of the term or terms you are looking for, such as deprecated, proposed, revised, deleted, etc. To use this option, choose a status from the drop down list. Note: if you do not choose anything from this list, only published concepts will be displayed.

  5. Classification scheme: using this option will restrict your search to concepts that are part of the chosen classification scheme, such as the AGRIS/CARIS Classification Scheme or the FAO Technical Knowledge Classification Scheme. To use this option, choose a scheme from the drop down menu.


After you have made your choice(s), click the search button to perform the search.
