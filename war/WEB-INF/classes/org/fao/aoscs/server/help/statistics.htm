Statistics Module

Administrators, publishers and validators have access to this module.
This module provides current information on users' country, login and logout time, duration of visits, use of language versions, and frequency of user actions.

The following is further explanation of the statistics available:

  • Languages refers to the display languages available to show the data and the number of concepts/terms available in that language.

  • Status refers to the current status of terms and concepts.

  • The user's section shows the number of proposed terms, etc. per user.

  • The relationship section shows the number of concepts associated with each relationship.
